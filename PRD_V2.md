## 一、项目概况

### 1.1 项目背景

湖北态禾农业股份有限公司从事桃树、梨树规模化种植及鲜果销售，拥有自主种植基地与稳定供应链，业务覆盖水果批发、商超供货、政企采购、终端零售等多个场景。

### 1.2 当前项目状态

**已完成阶段：**
- ✅ 前端基础架构搭建
- ✅ 核心页面实现（8个页面）
- ✅ 首页模块完善（轮播图、数据统计、产品展示、案例展示）
- ✅ 响应式设计实现
- ✅ 本地图片替换（轮播图、产品图）
- ✅ 交互动效优化

**技术栈调整：**
- 原计划：Vue3 + Vite + Element Plus
- 现实现：Next.js 16 + React 19 + TypeScript + Tailwind CSS + shadcn/ui

### 1.3 核心痛点

- 品牌曝光不足，区域外市场认知度低
- 获客效率低下，无标准化询盘承接端口
- 信任建立困难，难以快速核实企业资质
- 信息传递滞后，产能变化无法及时同步客户

### 1.4 项目目标

**短期目标（已达成部分）：**
- ✅ 完成企业官方网站开发
- ✅ 实现核心展示功能
- ✅ 提供响应式用户体验

**中期目标（待实现）：**
- 📋 上线3个月内，获取有效B端询盘≥50条
- 📋 上线6个月内，核心关键词进入百度搜索结果前3页
- 📋 网站整体转化率≥3%

**长期目标：**
- 搭建标准化客户询盘渠道
- 建立后台内容管理系统
- 实现数据统计与分析

- 框架：Next.js 16.2.4（App Router）

### 10.2 技术方案建议

**方案A：Headless CMS + Next.js（推荐）**
- CMS：Sanity.io 或 Strapi
- 优点：开发快速、维护成本低、生态完善
- 缺点：有订阅费用

**方案B：自建后台系统**
- 后端：Next.js API Routes + Prisma
